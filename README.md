# Character Hub
**Character Hub** — это веб-платформа на Flask, которая превращает локальные JSON-файлы персонажей в живое сообщество. Это связующее звено в экосистеме моих RPG-инструментов.

# 🔗 Экосистема проектов
Этот репозиторий является частью цепочки:

1. Character Creator (PyQt): Десктопное приложение для генерации и экспорта героев в JSON.

2. Battle Arena (Arcade): Игра, которая импортирует данные из этой экосистемы для сражений.

3. Character Hub (Текущий): Веб-витрина для публикации, хранения и обсуждения этих героев.

# Основные функции
1. Интерактивная лента: Просмотр карточек персонажей с их аватарами.

2. Детальные профили: При клике на карточку открываются полные характеристики (stats), список способностей (abilities) и HP.

3. Социальное взаимодействие: Система лайков и комментариев для обсуждения билдов и характеристик.

4. База данных: Хранение и структуризация всех загруженных героев через SQLAlchemy.

# 🛠 Технологический стек
* Backend: Python, Flask

* Database: SQLAlchemy

* Frontend: HTML5, CSS3 (JinJa2 templates)

* Data Format: JSON

# 📝 Пример формата данных персонажа

```
    "class": ["маг", "buff('hero', 'hp', 10, 5)"],
    "level": 9,
    "stats": {"dexterity": 2, "strength": 1, "intelligence": 6, "charisma": 7},
    "hp": 6,

        {

        },
    {"name": "Пинок", "effect": "target['hp'] -= 3", "description": "Сильно пинает врага."}],
    "image_b64": ...
}
```
